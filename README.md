## 勤之助リマインダー

タイムカード訂正を失くす為のデスクトップAppです。


### ダウンロード

製作中

### セットアップ

主に開発者向けです。

パッケージをインストールする。

```
npm install --sabve --save-dev
```


アプリケーションを作成する。

```
node_modules/.bin/electron-packager ./ kinnosuke_riminder --platform=darwin --arch=x64
```
